# Source Review Area

This folder is reserved for optional selected source files or broader private source access.

The initial review package provides:

- clean metrics,
- source structure,
- source manifest,
- architecture notes,
- public evidence links,
- and evaluation context.

Add source files only when needed for a specific private review context.

Do not add:

- environment files,
- secrets,
- provider keys,
- private traces,
- local caches,
- build outputs,
- unnecessary binaries,
- or private project data.
