# Coeus AI — Private Review Package

This private repository provides codebase evidence and source-level review material for **Coeus AI**, a native C++ project-aware AI desktop workspace.

The public Coeus repository contains the demo release, rendered documentation, and public evaluation reports. This private package adds source-tree evidence, clean local code metrics, private architecture notes, and optional selected source access.

## Included

- Clean `scc` codebase metrics.
- Source file manifest and codebase structure notes.
- Private codebase overview.
- Architecture notes.
- Turn pipeline notes.
- Retrieval/context notes.
- Telemetry/evaluation notes.
- Public evidence links.
- Optional selected source or broader source access depending on review context.

## Excluded

- Secrets and provider keys.
- Environment files.
- Build output.
- Generated caches.
- Private local traces.
- Private project data.
- Unnecessary binaries.
- Generated parser/static-asset noise.
